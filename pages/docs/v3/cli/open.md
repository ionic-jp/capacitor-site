# Capacitor CLI - cap open

指定したネイティブ IDE（iOS は Xcode、Android は Android Studio）でネイティブプロジェクトのワークスペースを開きます。

```bash
npx cap open <platform>
```

<strong>Inputs:</strong>

- `platform` (required): `android`, `ios`
